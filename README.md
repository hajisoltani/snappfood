# SnapFood Demo

A simple **SnapFood-inspired frontend demo** built with pure **HTML** and **CSS**.  
This project demonstrates a basic food delivery UI without any backend functionality.

---

## Features
- Homepage with featured restaurants
- Menu display for food items
- Responsive layout for desktop and mobile
- Simple CSS animations for hover effects
- Clean and modern UI inspired by SnapFood

---

## Tech Stack
- HTML5
- CSS3 (Flexbox & Grid)
- JS
- Responsive Design


---
